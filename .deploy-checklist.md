# Чеклист для развертывания на сервере

## Файлы для загрузки на сервер:

✅ Обязательные файлы:
- `bot.py` - основной файл бота
- `database.py` - работа с БД
- `requirements.txt` - зависимости
- `config.py.example` - шаблон конфигурации
- `.gitignore` - игнорируемые файлы

✅ Файлы для развертывания:
- `deploy.sh` - скрипт автоматического развертывания
- `income-bot.service` - systemd service файл
- `DEPLOY.md` - подробная инструкция

✅ Опциональные:
- `stop_bot.sh` - скрипт остановки бота (для локальной разработки)

❌ НЕ загружайте на сервер:
- `config.py` - создайте на сервере с вашим токеном
- `income_bot.db` - база создастся автоматически
- `__pycache__/` - кэш Python

## Быстрый старт:

1. Загрузите файлы на сервер
2. Запустите: `./deploy.sh`
3. Настройте `config.py` с токеном
4. Выберите способ запуска (systemd/screen/tmux)
5. Проверьте работу бота

Подробная инструкция в файле `DEPLOY.md`
